"# MiniERP_SYSTEM"

MiniERP_SYSTEM layihəsi kiçik bir ERP sistemidir. Bu layihə Spring Boot ilə hazırlanıb və aşağıdakı mikroservisləri ehtiva edir:

- Auth Service
- Inventory Service
- Order Service
- Notification Service
- Report Service
- API Gateway

## Texnologiyalar
- Java 17
- Spring Boot
- Maven
- JWT Authentication
- REST API
